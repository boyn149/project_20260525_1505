# Layer2: Conceptual Design - ถอดรหัส INFJ Door Slam: กลไกการตัดจบความสัมพันธ์และการตระหนักรู้ในตนเอง

**Pedagogy**
การเรียนรู้ผ่านการวิเคราะห์เชิงจิตวิทยาและทฤษฎีบุคลิกภาพ MBTI โดยเน้นการอธิบายกลไกการทำงานของฟังก์ชันทางจิตวิทยา (Cognitive Functions) เพื่อให้ผู้อ่านเข้าใจกระบวนการ สาเหตุ และแรงกดดันที่นำไปสู่การเกิดปรากฏการณ์ Door Slam อย่างเป็นระบบและมีเหตุผล มุ่งเน้นให้เกิดการตระหนักรู้ในตนเอง (Self-awareness) เพื่อให้สามารถรู้เท่าทันอารมณ์และขีดจำกัดของตนเองก่อนที่จะถึงจุดแตกหัก

**Writing Style**
ใช้รูปแบบการเขียนแบบ RWOP (Rich Text without pic) โดยมีรายละเอียดดังนี้:
* ใช้ตัวหนา (Bold) เพื่อเน้นจุดสำคัญของเนื้อหา
* ใช้ตัวเอียง (Italic) เพื่อเน้นความหมายหรือคำศัพท์เฉพาะทางเชิงจิตวิทยาและ MBTI
* ใช้ Bullet / List เพื่อจัดข้อมูลและลำดับกระบวนการคิดให้อ่านและทำความเข้าใจได้ง่าย
* มีการใช้ตาราง (Table) สำหรับเปรียบเทียบหรือสรุปข้อมูลสำคัญให้เห็นภาพรวมชัดเจน
* มี Callout Box แบบมีไอคอน (รูปแบบ >{icon} **{topic}** : {content}) เพื่อแทรกข้อความสำคัญ เกร็ดความรู้ หรือข้อควรระวัง
* **ไม่มีการแทรกรูปภาพประกอบในเนื้อหา** ตามเงื่อนไขของ RWOP
