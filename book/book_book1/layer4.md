# Layer4: Action - เสน่ห์เงียบฉบับ INFJ: ดึงดูดคนที่ใช่ด้วยศิลปะแห่งความสงบ

**Content Writing**
การเขียนเนื้อหาจะมุ่งเน้นไปที่การอธิบายเชิงจิตวิทยาและทฤษฎีบุคลิกภาพอย่างเป็นเหตุเป็นผล ใช้ระดับภาษาที่เป็นมืออาชีพ น่าเชื่อถือ และให้ข้อมูลที่ชัดเจนตรงไปตรงมา (ไม่ใช้สำนวนการพูดแบบบุคคลหรือเอสเธอร์) โดยเนื้อหาจะถูกเขียนตามรูปแบบ RWP (Rich Text with pic) ซึ่งมีแนวทางการนำเสนอดังนี้:
*   **การเน้นย้ำข้อความ:** ใช้ตัวหนา (Bold) เพื่อเน้นย้ำประเด็นหลักและข้อความที่สำคัญ ใช้ตัวเอียง (Italic) สำหรับคำศัพท์เฉพาะทางด้าน MBTI (เช่น *Ni, Fe, Ti, Se*) และคำศัพท์เชิงจิตวิทยาจาก *The Art of Seduction* (เช่น *The Siren, The Coquette*)
*   **การจัดระเบียบข้อมูล:** ใช้ Bullet และ Numbered List ในการอธิบายกลไกการทำงานของฟังก์ชันทางจิตวิทยา หรือขั้นตอนการสร้างแรงดึงดูด เพื่อให้อ่านง่ายและย่อยข้อมูลได้เร็ว
*   **การเปรียบเทียบและสรุป:** ใช้ตาราง (Table) ในการเปรียบเทียบหรือสรุปกลยุทธ์ เพื่อให้เห็นภาพความเชื่อมโยงระหว่าง MBTI และเทคนิค Passive Attractive
*   **การแทรกเกร็ดความรู้:** ใช้ Callout Box พร้อมไอคอน (รูปแบบ `>{icon} **{topic}** : {content}`) สำหรับแทรกเคล็ดลับเพิ่มเติมหรือข้อควรระวังในการนำทฤษฎีไปปรับใช้

**Visual**
การนำเสนอภาพประกอบจะสอดคล้องกับรูปแบบ RWP โดยเน้นไปที่ภาพประเภท Educational Infographic เพื่อช่วยเสริมความเข้าใจในเนื้อหาที่ซับซ้อนให้เห็นภาพได้ชัดเจนยิ่งขึ้น โดยมีแนวทางการกำหนด Prompt ดังนี้:
*   **รูปแบบและสัดส่วน:** กำหนดสัดส่วนภาพที่ 16:8 สไตล์การออกแบบเป็นแบบ Minimal เรียบง่าย สบายตา และใช้พื้นหลังสีขาว (White background) เพื่อให้ดูเป็นงานเชิงวิชาการและอ่านง่าย
*   **การใช้ภาษาในภาพ:** เขียนคำสั่ง Prompt เป็นภาษาอังกฤษ แต่จะมีการระบุเงื่อนไขอย่างเคร่งครัดว่าข้อความที่ปรากฏในภาพทั้งหมดจะต้องเป็นภาษาไทยเท่านั้น (All text in the image MUST be completely in Thai language)
*   **ประเภทของ Infographic ที่เลือกใช้:**
    *   *Informational Infographic:* สำหรับการอธิบายการทำงานของฟังก์ชันทางจิตวิทยาหลักและรองของ INFJ
    *   *Process Infographic:* สำหรับการแสดงขั้นตอนและกลไกการสร้างเสน่ห์หรือการใช้กลยุทธ์นักดึงดูดแบบไม่ต้องออกแรงไล่ตาม (Passive Attractive) อย่างเป็นลำดับขั้นตอน
