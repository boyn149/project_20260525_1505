# Philosophy 
ผู้หญิง INFJ มีเสน่ห์เฉพาะตัวที่เกิดจากฟังก์ชันทางจิตวิทยา (Ni, Fe, Ti, Se) อันลึกซึ้ง การผสานความรู้ด้านบุคลิกภาพ MBTI เข้ากับกลยุทธ์เชิงจิตวิทยาจากหนังสือ Art of Seduction จะช่วยให้สามารถดึงพลังความสงบ ความเห็นอกเห็นใจ และความลึกลับน่าค้นหา มาใช้เป็นแรงดึงดูดแบบไม่ต้องออกแรงไล่ตาม (Passive Attractive) ได้อย่างมีประสิทธิภาพและเป็นธรรมชาติ
# Objective
1. เพื่อให้รู้กลไกการสร้างเสน่ห์ของผู้หญิง INFJ แบบไม่ต้องเป็นฝ่ายรุก
2. เพื่อให้รู้ทันตัวเอง
3. เพื่อให้สามารถดึงดูดคนที่รักได้
# Constrain
1. เนื้อหาต้องมีการพูดถึง type ของ MBTI
2. จำนวน 1 บท 2 หัวข้อย่อย
3. มีการเอาเนื้อหาจาก Art of Seduction เข้ามาประกอบ
4. มีรูปภาพแทรกในเนื้อหา
5. ใช้รูปแบบการเขียนแบบ RWP (Rich Text with pic)

# Philosophy 
ปรากฏการณ์ Door Slam หรือการตัดความสัมพันธ์อย่างเด็ดขาดของ INFJ ไม่ใช่การใช้อารมณ์ชั่ววูบ แต่เป็นกลไกการป้องกันตนเองขั้นสุดท้ายที่ซับซ้อน ซึ่งเกิดจากการทำงานของฟังก์ชันทางจิตวิทยาที่แบกรับความกดดันและการถูกละเมิดขอบเขตมาอย่างยาวนาน การทำความเข้าใจกระบวนการเชิงลึกนี้จะช่วยสร้างความตระหนักรู้และทำให้สามารถเท่าทันสภาวะอารมณ์ของตนเองได้ดียิ่งขึ้น
# Objective
1. เพื่อให้รู้กลไกการ Door Slam ของ INFJ
2. เพื่อให้รู้ทันตัวเอง
# Constrain
1. เนื้อหาต้องมีการพูดถึง type ของ MBTI
2. จำนวน 1 บท 2 หัวข้อย่อย
3. ไม่มีรูปแทรก
4. ใช้รูปแบบการเขียนแบบ RWOP (Rich Text without pic)
